# Input
The program should take in a file named ```graph.txt```. This file takes the following format:
```text
    N M
    u0 v0
    u1 v1
```
Where:
    N = number of vertices
    M = number of edges
    ux = a starting vertex
    vx = a destination vertex
Example:
```text
    3 2
    1 2
    2 3
```
says there are 3 veritices and 2 edges and it creates a connection between verticies 1 and 2 and a connection between 2 and 3.
